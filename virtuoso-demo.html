Thinking, Fast and Slow.

Fast Thinking are cognitive processes that happen quickly and with no deliberation.
Slow Thinking are cognitive processes that require deliberation and concentration. For humans these are difficult, energy and time consuming processes.
Virtuoso uses this bimodal cognitive model as a routing system. 
              
                defmodule ProjectName do
                  @moduledoc """
                  Your bot's description.
                  """
                  alias ProjectName.FastThinking
                  alias ProjectName.SlowThinking
                  alias ProjectName.Routine

                  @recipient_ids [
                    Application.get_env(:project_name, :fb_page_recipient_id)
                  ]

                  @doc """
                  Getter for receiver_id
                  """
                  def recipient_ids(), do: @recipient_ids

                  @doc """
                  Main pipeline for the bot.
                  """
                  def respond_to(impression, conversation_state) do
                    impression
                    |> FastThinking.run()
                    |> SlowThinking.run()
                    |> Routine.runner(conversation_state)
                  end
                end

Routines

  • Satisfy intents.
  • Actuators - do the thing
  • Independent - can refer to one another
              
                defmodule YoBot.Routine.Greeting do
                  @moduledoc """
                  YoBot's greeting routine.
                  """

                  def run() do
                    "Yo."
                  end
                end
From start to finish we maintain the conversation state in a `Virtuoso.Conversation` Genserver Module so that routines can refer to previous parts of an exchange and maintain a sense of context.
